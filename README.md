# Superstore Sales Dashboard

## Project Overview

This project analyzes Superstore sales data using Excel, SQL, and Power BI to identify sales trends, profitability, customer behavior, and regional performance.

## Tools Used

- Microsoft Excel
- MySQL
- Power BI

## Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Quantity KPI
- Sales by Region
- Profit by Category
- Monthly Sales Trend
- Top 10 Customers
- Top 5 Products
- Sales Distribution by Category
- Interactive Filters

## Business Insights

- West Region generated the highest sales.
- Technology is the most profitable category.
- High discounts negatively impact profit.
- November recorded the highest sales.

## Files Included

- Power BI Dashboard (.pbix)
- SQL Queries
- Cleaned Dataset
