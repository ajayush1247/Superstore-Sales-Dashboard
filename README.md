# 📌 Project Overview

This project analyzes the Superstore Sales dataset using **Excel**, **MySQL**, and **Power BI** to uncover business insights and build an interactive dashboard.

The dashboard helps management understand:

- Sales performance
- Regional performance
- Product performance
- Customer behavior
- Profitability
- Sales trends

---

# 💡 Key Business Insights

### Highest Sales Region
- West Region generated the highest sales.

### Most Profitable Category
- Technology is the most profitable category.

### Discount Impact
- High discounts reduce profitability significantly.
- Discounts above 30% often result in losses.

### Best Performing Month
- November records the highest sales.
